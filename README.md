# The Weather Agent

A CLI weather agent built with LangGraph.js, Groq (Llama 3.3), and Open-Meteo.

## Requirements

- Node.js v18+
- A free [Groq API key](https://console.groq.com/keys)

## Setup

```bash
npm install
```

Copy `.env.example` to `.env` and add your key:

```bash
cp .env.example .env
```

```env
GROQ_API_KEY=gsk_...
```

## Run

```bash
npm start
```
